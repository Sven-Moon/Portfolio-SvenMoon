Project Overview

This site was designed to fill up screens large and small with mobile, desktop, and ultra-big desktop views. It allows the user to assign items to friends/family or remain anonymous. Orders are kept in history and favorites are used to start from prior orders (coming soon). Order completion can be tracked from the app. Users enjoy animations such as smooth closing/opening containers and a moving menu highlight. Common errors are handled efficiently using intercetpors. Many accessibility factors such as alternative text, input purpose, and contrast minimums are observed.

The app is presently hosted on firebase. I'm presently working on completion of a .NET webAPI that will handle authorization and persistent storage of orders.
